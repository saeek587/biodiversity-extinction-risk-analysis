## üîç Methodology Highlights
1. **Regression Analysis**: Cubic polynomial models to forecast 167,000+ species assessments by 2028
2. **Distribution Fitting**: Negative binomial model for country-level threatened species estimation
3. **Markov Chain Analysis**: One-step transition probability matrices for conservation status prediction
4. **Random Forest Classification**: Machine learning model to predict future conservation status
5. **Difference-in-Differences**: Policy evaluation of Wildlife Protection Act on tiger populations
6. **Primary Survey Analysis**: 300+ respondents, revealed 50% awareness gap in biodiversity conservation

## üìà Key Results
- Predicted **269,821 species assessments by 2028** (79% increase from 2022)
- Identified **71,522 threatened species by 2028** using cubic regression models
- Quantified **634 Bengal tiger population recovery** post-2002 Wildlife Act
- Achieved **52% classification accuracy** using Random Forest on multi-class conservation status
